Civil War Trivia, by Daniel Wilhelm

Created with XMedia Programming Library also by Daniel Wilhelm.


To answer a question, press:

Player 1
A - Select choice A
S - Select choice B
D - Select choice C
F - Select choice D

Player 2 (Numeric Keypad)
4 - Select choice A
5 - Select choice B
6 - Select choice C
Enter - Select choice D

Escape - Exits (slight delay)
